Как использовать CSS-генератор

Это расширение для платформы Twitter Bootstrap Генератор упрощает создание красивых кнопок..

Сначала используйте ползунки для выбора стиля.

Используйте клавиши со стрелками для дополнительной точности. (Button Puffiness может не повлиять на все браузеры.)

Во-вторых, скопируйте CSS в поле снизу.

Вы должны просто вставить его в свой CSS-файл. Примените класс “btn-custom” к любой кнопке (или другому элементу) на вашем сайте, чтобы вы хотели иметь эти цвета.

Примеры! (вы, вероятно, можете сделать лучше, чем эти, хотя...)
